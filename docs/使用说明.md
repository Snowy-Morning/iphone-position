# iPhone 虚拟定位 — 使用说明

通过 USB 修改 iPhone **系统级 GPS**，使「查找」、地图、位置共享显示虚拟坐标。

> **独立发布包，无需安装 Python。**  
> 解压到**纯英文路径**（如 `D:\PositionTool`），不要放在中文目录下。

> 需保持 USB 连接；断开或关闭工具后恢复真实定位。

---

## 首次配置（按顺序）

| 步骤 | 操作 |
|------|------|
| 1 | 双击 `install_apple_usb.bat` 安装 Apple USB 服务 |
| 2 | USB 连接 iPhone，解锁，点「信任此电脑」 |
| 3 | 双击 `download_ddi.bat` 下载开发者镜像 |
| 4 | 双击 `enable_developer_mode.bat`，手机开启开发者模式并重启 |
| 5 | 运行 `check_env.bat` 确认全部 OK |

## 每次使用

```
1. USB 连接 iPhone 并解锁
2. 右键 start_tunnel.bat → 以管理员运行（保持打开）
3. 双击 `start_app.bat`
4. 选点后点「应用定位」
5. 用完点「恢复真实定位」
```

## 自定义快捷地点

编辑同目录 `config.py` 中的 `PRESETS`，保存后重启工具。

## 常见问题

| 现象 | 处理 |
|------|------|
| 找不到设备 | 运行 `check_env.bat`，检查 USB / 信任 / 隧道 |
| 开发者模式未开 | `enable_developer_mode.bat` |
| DDI 失败 | `download_ddi.bat` 或开 VPN |
| 查找不更新 | 完全退出「查找」重开，等 1–3 分钟 |
